What is a variable?

#5

Variables are what allows the user to define what they want thier value to be. In C++, We have to explicitly define what each variable's type is using one of the folowing indicators.

int stands for integer and stores numberical values
char stands for character and stores characters such as 'A'.
string also known as str stores strings of text such as "What is your name?"

For example, if a user wished to store the number 5. They would use an int as it will allow them to store a numerical value.
If a user wished to store an name of a person, they would use a string as it allows them to store multiple characters.

A String is simply an array of characters
